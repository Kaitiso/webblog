# Audio Files for Touhou Mini Game

This directory should contain background music files for each stage.

## Required Files:

- **stage1.mp3** - Background music for Stage 1 (Marisa)
- **stage2.mp3** - Background music for Stage 2 (Elly)
- **stage3.mp3** - Background music for Stage 3 (Yuuka)

## File Format:
- Supported formats: MP3, OGG, WAV
- Recommended: MP3 for best browser compatibility
- File size: Keep under 5MB per track for faster loading

## Notes:
- Music will loop automatically during gameplay
- If files are missing, the game will continue without audio
- You can use royalty-free music or Touhou fan arrangements
- Make sure you have proper licensing for any music used

## Example Sources for Royalty-Free Music:
- YouTube Audio Library
- Free Music Archive
- Incompetech
- Touhou Wiki (for official/fan arrangements with proper attribution)
